PacMan (Java)

Harmadik féléves projektem a klasszikus PacMan játék elkészítése. A pálya, szellemek, PacMan grafikája adott volt, de a játékmenet lefejlesztését én csináltam.

Ruharaktár (C, C++, Java)

Ezt a projektemet 3 féléven keresztül újra és újra lefejlesztettem 0-ról, 3 különböző nyelven. A program célja egy adatbázis kezelése OO módon, mely a ruharaktárban tárolt jelmezekkel hajt végre különböző műveleteket. A programot a közeljövőben implementációra kész állapotban szeretném implementálni egy művészeti iskolában.

Spaceship

Szintén harmadik féléves projektem egy egyszerű félkész állapotó Spaceship játék tesztelése, majd javítása, kiegészítése.

Rólam

Harmadéves mérnökinformatikus egyetemista vagyok a BME-n. Tanulmányaim során C, C++ és Java nyelven szereztem jelentős tapasztalatot, Python nyelven autodidakta módon tanultam meg programozni. Szeretném tudásomat minél előbb kamatoztatni, tapasztalatot szerezni és egy szimpatikus munkahelyen hosszú távon elhelyezkedni.
